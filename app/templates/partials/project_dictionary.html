Project dictionary

Key-value labels used to fill document-template {{ '{{ tag }}' }} tags. Dictionary entries override template defaults when you render a .docx for this project.

How it works

Each entry has a key (must match a tag inside your Word file, e.g. part_number) and a value (what gets substituted, e.g. ABC-12345). Optional: a friendlier label for display, and notes.

Built-in tags like {{ '{{ project_name }}' }}, {{ '{{ today }}' }}, {{ '{{ milestones }}' }} are always available — you don't need to add them here.
